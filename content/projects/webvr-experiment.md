## Idea & Description

An experimental project focused on creating a basic virtual reality
experience directly in the web browser using the WebXR API and Three.js.
The goal was to understand the fundamentals of setting up a VR scene,
handling controller input, and basic interactions within a web
environment.

## Key Features / Learnings

- Integrated the WebXR API with Three.js for VR rendering.
- Implemented basic VR controller input handling, including teleportation
  and grabbing.
- Learned about VR considerations like frame rates and user comfort.
- Explored the potential of browser-based VR experiences.
